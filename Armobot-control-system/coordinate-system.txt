link lengths :

L1 : 24cm # height of base from ground 
L2 : 21 cm # length of joint 2
L3 : 35 cm # length of joint 3 including end effector

consider that shape of base is square with dimensions of 23.5 x 23.5 x L1 (L x B x H)

once calibration is done, then below scenarios will be considered for coordinate system of the robotic arm.
Consider joint 1 as shaft placed at the middle of this box. So bottom end of joint 1 should get considered as Origin of coordinate O(0,0,0). As you know shaft only perform rotation motion so in any case coordinate of bottom end will be always O(0,0,0).

Top end of joint 1 is connected to bottom end of joint 2 so coordinate of this point will be 1J2(0,0,24). this point neither gets affected by angular motion of joint1 nor joint2.

Top end of Joint 2 will be at 2J3(0,0,45) and this will get affected by angular motion of joint 2 and joint 1.

Bottom end of joint3 is connected with top end of joint 2 at point 2J3(0,0,45).
After calibration top end of joint3 will be at E(0,35,45).

now we fixed all movement points in our coordinate system, by this system now if we move base in left direction value of x coordinate in E will be negative and if we move base in right direction then value of X will be positive.

By now using geometry functions we can calculate value of coordinates 2J3 and E.
